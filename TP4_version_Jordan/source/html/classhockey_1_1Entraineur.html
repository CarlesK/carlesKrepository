Detailed Description

Cette classe sert au maintien et à la manipulation des entraineurs. Elle hérite de la classe Personne.

Les valeurs qui entrent dans la classe doivent être validée à priori.

Attributs: string p_nom, string p_prenom, Date p_dateNaissance, string p_telephone, p_numRAMQ, p_sexe

[in]p_telephoneun string représentant le numéro de téléphone de l'entraîneur
[in]p_numRAMQun string représentant le numéro de RAMQ de l'entraîneur
[in]p_sexeun char représentant le sexe de l'entraîneur
Precondition
Tous les paramètres en entrée doivent être valide pour une personne standard.
Le numéro de RAMQ de l'entraîneur doit être valide avec les infos de l'entraîneur et avoir un format valide
Le sexe de l'entraîneur doit être 'm' ou 'M' pour homme et 'f' ou 'F' pour femme.
Invariant
Les invariants de l'entraîneur sont les mêmes que ceux d'une personne standard.
L'entraîneur doit également avoir un sexe valide et un numéro de RAMQ valide.
