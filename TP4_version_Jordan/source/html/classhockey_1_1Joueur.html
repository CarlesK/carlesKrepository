Detailed Description

Cette classe sert au maintien et à la manipulation des joueurs. Elle hérite de la classe Personne.

Les valeurs qui entrent dans la classe doivent être validée à priori.

Attributs: string p_nom, string p_prenom, Date p_dateNaissance, string p_telephone, string p_position

Precondition
Tous les paramètres en entrée doivent être valide pour une personne standard.
La position du joueur doit être "centre","ailier","gardien" ou "défenseur"
Invariant
Les invariants d'un joueur sont les mêmes que ceux d'une personne standard.
Le joueur doit avoir une position valide.
